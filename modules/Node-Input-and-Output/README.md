# Node Input and Output

In this module you'll learn several command ways to get data into your node
program and several ways to generate output from your node program.

## Skills

- can read arguments
- can read from STDIN
- can read a file
- can write to STDOUT
- can write to STDERR
- can write to a file

```

```
